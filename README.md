# 🛠️ Page Audit DevTools

> یه snippet ساده که امنیت، performance، 
> accessibility و SEO صفحه رو در ۱ ثانیه آنالیز می‌کنه

![preview](docs/preview.png)

## ✨ Features
- 🔐 Security Check
- ♿ Accessibility Check  
- ⚡ Performance Check
- 🔎 SEO Check
- 📋 Form Check (مخصوص فین‌تک)

## 🚀 How to use
1. F12 بزن
2. Sources → Snippets
3. New Snippet → paste کن
4. Ctrl+Enter
